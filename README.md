# calculator
calculator function using tdd


## node version

```
nvm use v22.14.0
```

## using bun (1.1.42)

To create project

```
bun init 
```


## To install Dependencies
```
bun install
```

## To Compile and build
```
bun run build
```


## To Run Tests

```
bun test
```
or

```
bun run test
```
